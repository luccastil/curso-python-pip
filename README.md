# Game project

Para correr el juego debes seguir las siguientes instrucciones en la terminal:

```sh
cd game
python3 main.py
```

# App project

Para correr el juego debes seguir las siguientes instrucciones en la terminal:

```sh
git clone
cd app
source app-env/bin/activate
pip3 install -r requirements.txt
python3 main.py
```
